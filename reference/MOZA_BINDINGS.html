Notes. Operator mode (Guns / Missile / Scan / Mining / Salvage) is set on the MTQ mode knob (17–21); it decides which action a multi-mode button fires. Power management (WPN hat pips) and shield strength work on every ship — the old directional shield-faceting was removed (it only worked on Size-3-shield ships). On foot you use an Xbox controller, not this rig. Two bindings to confirm in-game: the WPN-hat power pips (v_engineering_assignment_*) and Pin Target (v_target_pin_selected) — if either does nothing, its action-map needs adjusting.
